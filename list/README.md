## **구현 명세**
> 추상데이터 타입 T\
> 각 메소드의 반환형은 구현언어의 명세에 따름
>### **push_front(T value)**
> 리스트의 맨 처음 부분에 노드 삽입.
>### **push_back(T value)**
> 리스트의 맨 끝 부분에 노드 삽입.
>### **pop_front()**
> 리스트의 맨 처음 부분에 노드 삭제.
>### **pop_back()**
> 리스트의 맨 끝 부분에 노드 삭제.
>### **insert(iterator pos, T value)**
> 특정 위치에 입력된 값을 가진 노드를 추가.
>### **erase(iterator pos)**
> 특정 위치에 노드를 삭제.

## **추후 논의할 것**
> * 구현 명세의 확장.
> * 알고리즘에서의 활용 방안.
